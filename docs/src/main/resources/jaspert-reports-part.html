For generating reports Mapfish Print uses Jasper Reports. Jasper Reports is a mature library report generating reports. From the website:

The JasperReports Library is the world's most popular open source reporting engine. It is entirely written in Java and it is able to use data coming from any kind of data source and produce pixel-perfect documents that can be viewed, printed or exported in a variety of document formats including HTML, PDF, Excel, OpenOffice and Word.

In order to use Jasper reports template(s) are created. A template contains branding, styling and layout of the report. It also contains parameters which indicate the values that need to be provided when the report is generated.
