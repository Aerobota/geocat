About MarLIN - Marine Laboratories Information Network

MarLIN, the CSIRO Marine and Atmospheric Research Laboratories Information Network, stores information about data within the CSIRO Division of Marine and Atmospheric Research. The directory is accessible to CMAR staff and other users via a world wide web (WWW) interface for searching. If the interface hangs or MarLIN data do not appear after a reasonable interval (e.g. 1-2 hours), please notify us of any prolonged downtime in case we are not already become aware of the problem.

MarLIN entries include metadata elements which are compatible with the Marine Community Profile of the ISO19115/19139 metadata standard. MarLIN also incorporates keywords from a number of Thesauri - including:

  • NASA GCMD Earth Science Keywords (extended for CMAR use).
  • ANZLIC Search Words.
  • ANZLIC Geographic Extent Names (extended for CMAR use).
  • Codes for Australian Aquatic Biota (CAAB)

This catalogue can be accessed programmatically using the following services:

  • Catalogue Service for the web (CSW) de l'OGC
  • Open Archive Initiative (OAI-PMH)
  • Z39.50
  • OpenSearch
